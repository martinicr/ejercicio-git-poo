Ejercicio Git
=============

## Crear un repositorio local
```
git init <ruta>
git init .
```

## Annadir archivo
```
git add <ruta | expression regular>
git add README.md
git add README.md archivo.txt
git add *.md
git add .
```

## Confirmar los cambios
```
git commit -m <comentario>
git commit -m "Initial commit"
```

## Ver el estado del repositorio
```
git status
git log
git show
```

## Ver diferencias
```
git diff
```
